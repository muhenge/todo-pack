## Setup and run the app

- Run `git clone https://github.com/muhenge/todo-pack.git` in your terminal
- cd into the folder
- Run `npm install`
- Run `npm run build`
- Run `npm run start`
- Wait or open the app in your browser to see the app (http://localhost:8080)
